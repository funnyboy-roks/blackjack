# Blackjack

This is a simple blackjack game in your browser.

## Developing

To run the development server, run

```sh
npm install
npm run dev
```

or whatever equivalent for your preferred package manager/runtime.

The site is now running at <http://localhost:3000>

## Features

- Almost full set of rules (based on [this page](https://bicyclecards.com/how-to-play/blackjack/))
    - Missing splitting and insurance
- Local Multiplayer
- Betting
